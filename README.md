# User Context and LLM Response Quality Evaluation

This repository contains the datasets, survey results, model responses, and source code used in the study titled:

**“User Group and Context-Dependent Evaluation of LLM Response Quality.”**

## Project overview

The study investigates how user experience, prompt structure, and additional task-specific context influence the quality of responses generated by large language models.

The repository supports the reproducibility of the experiments and enables further analysis of the collected data.

## Files

- **`questions.xlsx`** – questions and prompt variants used in the experiments;
- **`responses.xlsx`** – responses generated by the evaluated language models;
- **`survey_responses.xlsx`** – anonymized responses collected during the user study;
- **`model_comparison.py`** – Python script used to calculate text similarity and evaluation metrics.

## Evaluation metrics

The analysis script calculates the following metrics:

- cosine similarity;
- Jaccard similarity;
- response length;
- number of unique tokens;
- BLEU;
- ROUGE-1, ROUGE-2 and ROUGE-L;
- BERTScore precision, recall and F1 score.

The script also generates heatmaps comparing the evaluated models.

## Installation

Install the required Python packages:

```bash
pip install -r requirements.txt
```

Install the Polish spaCy language model:

```bash
python -m spacy download pl_core_news_sm
```

## Running the analysis

Run the evaluation script from the main repository directory:

```bash
python src/model_comparison.py
```

Generated metrics and visualizations will be saved in the `results` directory.

## Data privacy

The survey data included in this repository have been anonymized and do not contain information that directly identifies study participants.
